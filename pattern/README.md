## Special Patterns

### 1. Butterfly Pattern
```
*                    *

*  *  *        *  *  * 
*  *  *  *  *  *  *  * 
*  *  *  *  *  *  *  * 
*  *  *        *  *  * 
*  *              *  * 
*                    *
```
**File:** `ButterFly.cpp`

---

### 2. Dubangg Pattern (Hourglass-like)
```
1  2  3  4  4  3  2  1  
1  2  3  *  *  3  2  1
1  2  *  *  *  *  2  1
1  *  *  *  *  *  *  1
```
**File:** `Dubangg.cpp`

---

### 3. Hollow Diamond Pattern
```
           * 
        *     *
     *           *
  *                 *
     *           *
        *     *
           *
```
**File:** `HollowDiamond.cpp`

## How to Run

1. Compile the C++ file:
   ```bash
   g++ filename.cpp -o output
   ```

2. Run the executable:
   ```bash
   ./output
   ```

3. Enter the value of `n` when prompted.
